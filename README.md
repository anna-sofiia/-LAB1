# -LAB1
Лабораторна робота №1 «Розпізнавання тексту із зображення»
У цій лабораторній роботі було використано датасет EMNIST для розпізнавання рукописних літер.
Зокрема, застосовано такі файли:

emnist-letters-train-images-idx3-ubyte — зображення для навчання
emnist-letters-train-labels-idx1-ubyte — мітки для навчання
emnist-letters-test-images-idx3-ubyte — зображення для тестування
emnist-letters-test-labels-idx1-ubyte — мітки для тестування

Датасет містить зображення рукописних літер англійського алфавіту, які використовуються для навчання нейронної мережі та перевірки її точності.
